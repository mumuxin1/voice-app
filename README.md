### VUE2.0_DEMO

## 技术栈
  vue2.0
  vuex
  vue-router
  axios
  mint-ui

## 目录结构

	├─index.html 								// 入口html文件
	├─package.json 								// 配置文件
	├─static								// 静态文件
	├─src									// 源码目录
	|  ├─App.vue 								// 页面入口文件
	|  ├─main.js 								// 程序入口文件，加载各种公共组件
	|  ├─style 								//公共样式文件
	|  ├─store 								// vuex的状态管理
	|  |   └index.js 									
	|  ├─service 								//数据交互统一调配
	|  ├─router 								//路由配置
	|  |   └index.js
	|  ├─plugins 								//引用插件
	|  ├─page 								//页面组件
	|  ├─images 								//公共图片
	|  |   └logo.png
	|  ├─config 								//基本配置
	|  ├─components 							//公共组件
	|  |     ├─first.vue
	|  |     └Hello.vue
	├─config 								//项目打包路径
	├─build 								//webpack配置
	|   ├─build.js 										
	|   ├─check-versions.js 							
	|   ├─dev-client.js 								
	|   ├─dev-server.js 								
	|   ├─utils.js 										
	|   ├─vue-loader.conf.js 							
	|   ├─webpack.base.conf.js 							
	|   ├─webpack.dev.conf.js 							
	|   └webpack.prod.conf.js 							


## 运行项目
  
  1. npm install
